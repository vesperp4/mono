# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

**Do not open a public issue.**

Instead, email the maintainers at the addresses listed in the [CODEOWNERS](.github/CODEOWNERS) file, or use [GitHub's private vulnerability reporting](https://github.com/vesperp4/mono/security/advisories/new).

We will acknowledge receipt within 48 hours and provide a timeline for a fix.
